# resonanceapp

Wij maken de Music Magnet Magazine. Het is de bedoeling dat wij een website (en smartphone app) maken waarin het mogelijk is om drie dingen te doen: Lezen over muziek, luisteren naar muziek EN het bekijken van muziekvideo's. Dit moet een aantrekkelijke muziekervaring creeëren voor jongeren. 

De doelgroep zijn jongeren van 14-25 jaar die heel erg veel van muziek houden en dus geïnteresseerd zijn in festivals en nieuwe muziekervaring ontdekken.
